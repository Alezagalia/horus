# US-133: Pantalla de Gestión de Rutinas (Mobile)

**Tipo:** user-story
**Prioridad:** high
**Sprint:** 14
**Story Points:** 6
**Asignado a:** Developer 2
**Estado:** todo

---

## Descripción

**Como** usuario móvil
**Quiero** crear y gestionar rutinas de entrenamiento
**Para** organizar mis entrenamientos en plantillas

---

## Criterios de Aceptación

### RoutinesScreen

- [x] Lista de rutinas con: nombre, descripción, X ejercicios, última vez, veces ejecutada
- [x] Botón "Iniciar" destacado
- [x] Long press: Editar, Duplicar, Eliminar

### RoutineFormScreen

- [x] Campos: nombre, descripción
- [x] Lista de ejercicios con drag-to-reorder
- [x] Cada ejercicio: nombre, target (3 × 10 @ 60kg), descanso
- [x] Botón "Agregar Ejercicio" → modal selector
- [x] Configurar targetSets, targetReps, targetWeight, restTime por ejercicio

### RoutineDetailScreen

- [x] Header con nombre, descripción
- [x] Botón principal "Iniciar Rutina"
- [x] Lista completa de ejercicios
- [x] Historial (últimas 5 ejecuciones)
- [x] Botones: Editar, Eliminar
