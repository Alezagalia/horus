# US-136: Pantalla de Estadísticas y Progreso (Mobile)

**Tipo:** user-story
**Prioridad:** high
**Sprint:** 14
**Story Points:** 5
**Asignado a:** Developer 2
**Estado:** todo

---

## Descripción

**Como** usuario móvil
**Quiero** ver mis estadísticas de progreso y evolución
**Para** analizar mi rendimiento y motivarme

---

## Criterios de Aceptación

### StatsScreen - Tab "General"

- [x] Selector de periodo: 7 / 30 / 90 días
- [x] Cards de métricas:
  - Entrenamientos Completados (+ mini gráfico)
  - Frecuencia (X/semana)
  - Volumen Total (+ gráfico evolución)
  - Duración Promedio
- [x] Top 5 ejercicios más realizados
- [x] Distribución por grupo muscular (gráfico dona)
- [x] Gráfico de frecuencia semanal (barras)

### StatsScreen - Tab "Por Ejercicio"

- [x] Selector de ejercicio
- [x] Selector de periodo: 30 / 90 / 365 días
- [x] Cards: Récord Personal, Progreso (+X kg), Veces Realizado, Volumen
- [x] Gráfico evolución peso máximo (línea)
- [x] Gráfico volumen por sesión (barras)
- [x] Último workout con series
